Primary Hosting

WaveWarz is optimized for deployment on Vercel with the frontend directory as the root. GitHub Pages now serves this lightweight static overview to avoid missing-page errors.

If you prefer Pages, the site published from the docs/ folder will appear at: https://candytoybox.github.io/wavewarz-base after the Pages workflow runs.

How GitHub Pages Deployment Works

  1. GitHub Actions uploads the docs/ folder as the Pages artifact.
  2. The Pages deployment step publishes that artifact to the github-pages environment.
  3. Source should be set to GitHub Actions in repository settings (the workflow configures this automatically on first run).

Trigger a deploy by pushing to main or running the Pages workflow manually.
